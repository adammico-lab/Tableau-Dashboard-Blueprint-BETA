## Quick Start

### Prerequisites

- [Claude Desktop](https://claude.ai/download) with agent mode enabled
- [Tableau MCP](https://github.com/tableau/tableau-mcp) installed and authenticated
- Access to a Tableau Cloud or Tableau Server site

### Install

1. Clone or download this repository.
2. Copy the skill folder into your Claude skills directory.
3. Restart Claude Desktop.
4. Confirm Tableau MCP is connected (you should see your site's content when you ask Claude about it).

### Example Prompts

```
Help me design a dashboard for my "Customer Retention" datasource. The audience is our CS leadership team and they want to know where churn is happening and why.
```

```
I have sales data with orders, products, regions, and dates. I need an executive dashboard my VP checks weekly. What should I build?
```

```
Wireframe a dashboard for our support team — they need to see open tickets by priority, SLA status, and agent workload. They check it throughout the day on desktop.
```

### What You Get

A complete design specification including: recommended chart types for each business question, dashboard layout wireframe, filter strategy, color palette, KPI hierarchy, interaction model (tooltips, drill-downs, actions), and a build sequence telling you which sheets to create first. Calibrated to your audience's time budget and decision-making context.
